# psau-certverify-app-issueance-generation

Next.js app for certificate verification and issuance.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Copy `.env.example` to `.env.local` and set real values (never commit `.env.local`).

## Deploy

Configure environment variables on your host (e.g. [Render](https://render.com)) — see `render.yaml` for keys. Use the dashboard for secrets such as `MONGODB_URI` and API keys.
